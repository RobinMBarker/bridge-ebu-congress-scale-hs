# Calculate Green Points for the EBU Congress Scale


## Modules

- GreenPoint: calculates green point scale

- GroupAwards: deals with ties in ranking list

## Executables

Compiled with `ghc --make awards` etc.

- awards: green master point awards as integers

-check: test GreenPoint

- greens: green point awards as decimals

- groupgreens: green points awards for ranks with ties
